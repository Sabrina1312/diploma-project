# 📄 Проект: Бюро переводов «Языки мира» — фронтенд-часть

## 📝 Общее описание

**Официальный сайт бюро переводов «Языки мира»**

Проект представляет собой **фронтенд-часть веб-сайта**, включающую:

- Калькулятор стоимости перевода документов
- Форму обратной связи с валидацией полей
- Загрузку файлов
- Адаптивное мобильное меню (бургер)
- Попапы (модальные окна)
- Динамическую подстановку инициалов в аватары отзывов
- «Ленивую» загрузку карты (iframe)

> **Важно:** В коде нет серверной логики. Вся обработка данных имитируется и выводится в консоль браузера.

---

## 🧩 Основные JavaScript-модули

| Модуль                  | Описание                                                                                                           |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------ |
| **Validator**           | Проверка email, телефона (10-11 цифр). Отображение/удаление ошибок.                                                |
| **FileManager**         | Прикрепление файла через скрытый `<input>`, отображение имени/размера, кнопка удаления.                            |
| **PopupManager**        | Открытие/закрытие попапов. Закрытие по клику на оверлей или кнопку.                                                |
| **Loader**              | Показ анимации загрузки при отправке. Управляется классом `.loader-visible` на элементе `#loader`.                 |
| **PriceCalculator**     | Расчёт стоимости перевода: базовая цена (700 руб./стр.), наценки за тип документа, нотариальные услуги, срочность. |
| **LanguageManager**     | Выбор языка «с» и «на». Ограничение: один из языков должен быть русским.                                           |
| **ExclusiveCheckboxes** | Взаимоисключающие чекбоксы (нотариальное заверение / копия).                                                       |
| **DataSync**            | Синхронизация данных из калькулятора в форму отправки (попап).                                                     |
| **UserForm**            | Сбор и очистка данных формы (имя, фамилия, email, телефон, сообщение, согласие).                                   |
| **OrderManager**        | Имитация POST-запроса (задержка 1.5 сек.). Вывод данных в консоль, прелоадер, попап успеха.                        |

## 🎨 Особенности интерфейса

### Адаптивность

- Мобильное меню (бургер) с блокировкой скролла при открытии
- Динамическая замена изображений в бегущей строке (`scroll-line`) под разрешения экрана

### Визуальные эффекты

- Hover-эффекты на главной странице (изменение цвета фона и текста)
- Подсветка текста при появлении в области видимости (Intersection Observer + задержка 1 сек.)

### Карта

- iframe загружается только после загрузки страницы (атрибут `data-src` → `src`)

### Аватарки

- Генерация инициалов из `data-name`
- Случайный цвет фона из массива

## ⚙️ Используемые технологии

- **HTML5** — семантическая разметка, формы, попапы
- **CSS3** — адаптив, флексбоксы, анимации, `clip-path`
- **JavaScript (ES6+)**
  - Модульная структура (объекты-синглтоны)
  - Работа с DOM, событиями, формами
  - Intersection Observer
  - LocalStorage (для управления cookie)
  - Имитация асинхронных запросов (async/await)

## 🍪 Управление согласием на cookie

- Баннер запрашивает разрешение на использование cookies
- Согласие сохраняется в `localStorage` на 365 дней
- При повторном визите баннер не показывается

---

## 🚀 Как запустить

1. Скачайте все файлы проекта.
2. Откройте `index.html` в любом современном браузере.
3. Для просмотра логов откройте **F12 → Console**.

> **Важно:** Все запросы к серверу имитируются, реальная отправка данных не происходит.
